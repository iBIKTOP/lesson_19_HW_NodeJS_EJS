Этот пример сайта создан на NodeJS с использованим шаблонизатора EJS.
Хранение данных реализованно в помощью файла JSON.
Допольнительно реализованна отправка письма на почту (обязательно необходимо указать свои почтовые данные)

Файлы предоставленны без модулей, используйте npm install для их подгрузки.

Также предоставленны скриншоты для ознакомления.
